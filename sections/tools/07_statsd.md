# StatsD

* Receive metrics via network (UDP/TCP)
* Aggregate metrics
* Transport metrics to one or multiple backends
* No collection
* No rendering
* Language: NodeJS

StatsD is most interesting for developers since it is very easy to embed code which sends metrics to a central StatsD.
